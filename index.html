Understand Money

Money is a tool for store of value and medium of exchange.

  • Store of value: you work, you create value, you need a tool that stores the value
  • Medium of exchange: Someone else creates something of value or provides a valuable service, you need a tool to exchange your value for their value

Monetary traits (a list of properties) determine if an object/item/device/thingamajig can function as this tool (as money).

Satoshi Nakamoto understood these traits and engineered Bitcoin.

Bitcoin is the greatest implementation of these traits into a monetary system.

Its dead simple: Bitcoin is the best money because it was designed to be the best money.
